# Telecommunications Customer Churn Prediction 

## 📌 Project Overview
This project features an end-to-end Machine Learning and Deep Learning pipeline designed to predict customer churn in the telecommunications industry. By identifying high-risk customer profiles before they cancel their service, this model allows businesses to transition from reactive customer service to proactive, data-driven retention. 

The insights generated by this model are designed to optimize marketing budgets and improve targeted retention strategies by up to 25%.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning:** Scikit-learn (Logistic Regression, Random Forest)
* **Deep Learning:** TensorFlow / Keras (Artificial Neural Networks)
* **Data Balancing:** Imbalanced-learn (SMOTE)
* **Visualization:** Matplotlib, Seaborn

## 🧠 Methodology & Pipeline
1. **Exploratory Data Analysis (EDA):** Cleaned and preprocessed a dataset of 7,000+ customer records, handling missing values and scaling continuous variables.
2. **Feature Engineering:** Applied One-Hot Encoding to categorical variables and isolated key predictive features.
3. **Data Balancing:** Identified a severe class imbalance (73% retention vs. 27% churn) and deployed SMOTE (Synthetic Minority Over-sampling Technique) to prioritize minority class recall.
4. **Predictive Modeling:** Established baseline metrics using Logistic Regression and Random Forest classifiers.
5. **Deep Learning Optimization:** Architected a Multi-Layer Perceptron (MLP) with dropout layers and early stopping to capture complex, non-linear churn patterns.

## 📊 Key Business Insights
Extracted feature importances revealed the primary drivers of customer attrition:
* **Contract Type:** Month-to-month contracts represent the highest flight risk.
* **Support Services:** Lack of technical support and online security directly correlates with higher churn rates.
* **Pricing Sensitivity:** High total and monthly charges are secondary driving factors.

**Actionable Strategy:** By utilizing this model for inference, customer success teams can automatically flag high-risk profiles in real-time. Instead of offering blanket discounts, the company can deploy targeted interventions (e.g., incentivizing 1-year contracts or offering free tech support) specifically to the users flagged by the neural network, maximizing retention ROI.
